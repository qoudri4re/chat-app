### CHAT APP

A chatapp created with the MERN stack and Material UI.

### features

1. Login and signup.
2. Sending and recieving messages.
3. Search for registered users.
4. Settings.
5. File upload (audio, video, image).
6. Delete and edit sent messages.
7. Unread messages count.
